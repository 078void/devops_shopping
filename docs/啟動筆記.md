## 本地
```text
cd shopping/Shopping.API
dotnet run
```
```text
cd shopping/Shopping.Client
dotnet run
```
## docker
```text
cd shopping
# 重新建置映像檔並啟動
docker-compose up -d --build

# 或分開執行
docker-compose build
docker-compose up -d
```
### 
```text
# 查看所有服務的日誌
docker-compose logs -f

# 只查看 API 的日誌
docker-compose logs -f shopping-api

# 只查看 Client 的日誌
docker-compose logs -f shopping-client

# 只查看 MongoDB 的日誌
docker-compose logs -f shopping-mongodb

```
前端網站：http://localhost:5001
後端 API：http://localhost:5000
